Array Functions

This is a collection of a array functions that I put together.

The included functions are as follows

  • isUniform - ensure that all the items in the array are uniform and returns a boolean value.
  • sum - adds up all the values in the array and returns the result
  • max - picks out the maximum value in the array and returns it
  • average - takes all the numbers in the array and returns the average.

You can see example runs in the console window.
